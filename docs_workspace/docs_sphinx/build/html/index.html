laygo2: Automating Custom Analog/Mixed-Signal/Digital IC Layouts#

[Source Code] [Colab Tutorial]

The LAYout with Gridded Object 2 (laygo2) package is a comprehensive framework for IC layout generation, written in the Python programming language.

The open-source version of laygo2 is currently being maintained by the Nifty Chips Laboratory at Hanyang University.

Its closed-source version requires a technology-transfer contract to be provided. Contact for details.
