class laygo2.object.physical.PhysicalObject(xy, name=None, params=None)[source]#

Bases: object

Base class for physical layout objects with physical coordinate info.

Notes

(Korean): 물리 객체들의 기본 클래스, 물리 좌표 정보를 갖고 있다.
